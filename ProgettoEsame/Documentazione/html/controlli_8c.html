Detailed Description

Questa libreria contiene la funzione di controllo da file.

La funzione "controllodafile" permette di verificare l'apertura dei file contenenti le matrici.

Version:
1.0
Date:
15/11/2022
Authors:
Balde Omar, Cutrignelli Vanessa, De Santis Samantha

La funzione "controllodafile" legge il valore della variabile "controllo", presente nella funzione "acquisizionefile", e ne assume lo stesso valore. La funzione viene usata dai vari case del "menu" nei casi in cui l'utente scelga di operare sulle matrici prima della loro acquisizione.

Parameters:
[in]controllo:int*puntatore a variabile intera
Returns:
valore di controllo

DIFENSIVA: supporta programmazione difensiva per la mancata acquisizione da file.
